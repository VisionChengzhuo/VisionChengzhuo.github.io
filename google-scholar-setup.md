# Google Scholar update checklist

目标：把 Scholar 主页调整为更像瑞川的展示方式，并把新的个人主页链接放进去。

1. 打开你的 Google Scholar 主页并登录：
   https://scholar.google.com/citations?user=dvG1nhgAAAAJ&hl=zh-CN&oi=ao

2. 点击头像/姓名旁边的铅笔编辑按钮，建议这样填：
   - Name: `Chengzhuo Tong, Peking University`
   - Affiliation: `Peking University`
   - Email for verification: 优先使用 `pku.edu.cn` 邮箱验证
   - Homepage: `https://visionchengzhuo.github.io/`

3. 如果你希望更正式一点，也可以把 Name 保持为 `Chengzhuo Tong`，只把 Affiliation 改成 `Peking University`。Google Scholar 通常会把单位显示在姓名下方，这种写法更常见。

4. 在个人主页部署完成后，再回 Scholar 把 Homepage 确认为最终公开链接。如果不用 `visionchengzhuo.github.io`，就把它替换成你的真实域名。
