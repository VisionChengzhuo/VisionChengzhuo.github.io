# Chengzhuo Tong Homepage

This is a static personal homepage designed for GitHub Pages.

## Preview

Open `index.html` directly in a browser.

## Deploy to GitHub Pages

1. Create or open the repository `VisionChengzhuo.github.io`.
2. Copy everything in this folder to the repository root.
3. Commit and push to the `main` branch.
4. The public homepage should become available at:
   `https://visionchengzhuo.github.io/`

After deployment, put that URL into the Homepage field of your Google Scholar profile.
